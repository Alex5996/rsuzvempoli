# 📅 RSU Turni

Applicazione Android dedicata alla gestione e consultazione delle informazioni RSU.

L'app nasce per raccogliere in un unico posto strumenti utili per i lavoratori, permettendo di consultare i propri turni, ricevere comunicazioni, accedere ai documenti e avere una bacheca sempre aggiornata.

---

# ✨ Funzionalità principali

## 📆 Calendario turni

- Visualizzazione calendario mensile
- Consultazione giornaliera dei turni
- Supporto squadre 1-5
- Memorizzazione della propria squadra
- Cambio squadra dalle impostazioni
- Evidenziazione del giorno corrente
- Navigazione tramite pulsanti e swipe

### Turnazioni supportate

- Ciclo 3-2
- Ciclo 6-2
- Gestione transizioni tra cicli
- Gestione periodi particolari
- Gestione riposi, smontanti e ferie

---

# 🏠 Home Screen

La schermata principale permette un accesso rapido alle funzioni più utilizzate.

Include:

- Widget "Il mio turno"
- Visualizzazione del turno odierno
- Indicazione della squadra selezionata
- Accesso rapido al calendario

---

# 📢 Bacheca comunicazioni

Sistema di comunicazione centralizzato tramite Firebase.

Funzionalità:

- Visualizzazione comunicazioni aziendali/RSU
- Aggiornamento in tempo reale tramite Firestore
- Pubblicazione di nuovi messaggi senza aggiornare l'app
- Gestione contenuti dinamici

---

# 📄 Documenti

Sezione dedicata alla consultazione dei documenti.

Funzionalità:

- Archivio documenti
- Visualizzazione elenco file disponibili
- Accesso rapido ai materiali condivisi
- Gestione documenti tramite database remoto

---

# 📝 Azioni e iniziative

Sezione dedicata alla gestione delle attività RSU.

Funzionalità:

- Visualizzazione delle azioni disponibili
- Informazioni sulle iniziative
- Aggiornamento dinamico dei contenuti tramite Firebase

---
