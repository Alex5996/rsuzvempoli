# 📅 RSU ZV Empoli

Applicazione Android per la consultazione rapida dei turni di lavoro.

RSU Turni nasce con l'obiettivo di avere sempre a portata di mano il proprio calendario turni, con una visualizzazione semplice, veloce e personalizzata in base alla propria squadra.

---

## ✨ Funzionalità principali

### 📆 Calendario turni

- Visualizzazione mensile dei turni
- Navigazione tra i mesi disponibili
- Visualizzazione giornaliera completa
- Indicazione del tipo di turno:
  - Mattina
  - Pomeriggio
  - Notte
  - Smontante
  - Riposo
  - Ferie

### 👷 Gestione squadra

- Selezione iniziale della propria squadra
- Memorizzazione automatica della scelta
- Possibilità di modificare o resettare la squadra dalle impostazioni

### 🏠 Home Screen

- Widget "Il mio turno"
- Visualizzazione immediata del turno odierno
- Indicazione della squadra selezionata
- Accesso rapido al calendario completo

### 🎨 Personalizzazione

- Supporto tema:
  - Automatico
  - Chiaro
  - Scuro

- Colori dedicati per ogni tipologia di turno:
  - Mattina
  - Pomeriggio
  - Notte
  - Smontante
  - Riposo
  - Ferie

### 🔄 Gestione turnazioni

Supporto alle rotazioni:

- Ciclo 3-2
- Ciclo 6-2
- Periodi di transizione tra cicli
- Gestione delle variazioni temporanee della turnazione
